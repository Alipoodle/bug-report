Finding your Discord client version

      

        
Android/iOS

        

          
    
            
  • Open the Settings menu and scroll down
    • 
            
  • The version number should be listed at the bottom or next to the 'Change Log' entry
    • 
            
  • Also include if you're using a Testflight (inc. build number) or Alpha/Beta build
    • 
          

        

        
Windows/Linux

        

          
    
            
  • Open %appdata%\discordcanary (Windows) or ~/.config/discordcanary (Linux)
    • 
            
  • The folder with the highest number is your version number
    • 
            
  • If using the PTB/stable clients, check the 'discordptb' or 'discord' folders instead
    • 
          

        

        
macOS

        

          
    
            
  • Click 'Discord Canary' (or similar) on the menu bar
    • 
            
  • Click 'About Discord'
    • 
          

        

        
Desktop Build Numbers

        

          
    
            
  • Open Dev Tools using Ctrl+Shift+I (Windows/Linux) or Cmd+Opt+I (macOS)
    • 
            
  • Click the Console tab
    • 
            
  • Scroll up and find the entry beginning '[BUILD INFO]'
